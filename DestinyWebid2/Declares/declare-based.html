Esta es una base para un declare con valores personalizables, que también contiene algo de HTML y CSS.

El editar la configuración es muy simple, solamente agregamos los valores necesarios en config(), el valor bg:LINK si queremos un fondo, y header:COLOR HEXADECIMAL para editar la cabecera. Tiene incluida una imagen entre medias para, además de ser estetico, poder desplegar y ver el fondo lo más completo posible.
